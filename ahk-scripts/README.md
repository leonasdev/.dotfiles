# AHK Scripts

Personal AutoHotkey v2 scripts for Windows automation.

## Scripts

### MediaControl.ahk

Control Spotify and Pocket Casts playback via global hotkeys.

| Hotkey | Action |
|--------|--------|
| Ctrl+Win+S | Toggle between Spotify / Pocket Casts |
| Ctrl+Win+P | Play / Pause |
| Ctrl+Win+J | Skip next |
| Ctrl+Win+K | Skip previous |
| Ctrl+Win+L | Fast forward (Spotify) / Skip next (Pocket Casts) |
| Ctrl+Win+H | Rewind (Spotify) / Skip previous (Pocket Casts) |

## Requirements

- Windows 10/11
- [AutoHotkey v2](https://www.autohotkey.com/)
